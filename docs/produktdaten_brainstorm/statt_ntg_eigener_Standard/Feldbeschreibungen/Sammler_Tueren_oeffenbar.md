# Tueren_oeffenbar

## Feldgruppe
Sammlermodelle

## Beschreibung
Angabe, ob die Türen des Modells zu öffnen sind.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Tueren_oeffenbar |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Öffenbare Türen sind ein wichtiges Funktionsmerkmal bei Sammlermodellen und ermöglichen den Blick auf das detaillierte Interieur.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Türen sind zu öffnen |
| N | Türen sind nicht zu öffnen / nicht vorhanden |

## Beispiele
| Qualitätsstufe | Tueren_oeffenbar |
|----------------|------------------|
| Museum Quality | J |
| Premium | J |
| Standard | J/N (variiert) |
| Spielzeug | meist N |
