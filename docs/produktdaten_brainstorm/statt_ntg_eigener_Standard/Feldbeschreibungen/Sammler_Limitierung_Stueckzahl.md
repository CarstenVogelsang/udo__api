# Limitierung_Stueckzahl

## Feldgruppe
Sammlermodelle

## Beschreibung
Die Stückzahl bei limitierten Editionen.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Limitierung_Stueckzahl |
| Datentyp | Ganzzahl |
| Pflichtfeld | Nein |
| Minimalwert | 1 |
| Maximalwert | 999999 |

## Hinweis für Datenlieferanten
Die Limitierung ist ein wichtiger Wertfaktor für Sammler. Je niedriger die Auflage, desto höher typischerweise der Sammlerwert.

**Typische Auflagen:**
- Extrem limitiert: 50-500 Stück
- Stark limitiert: 500-2.000 Stück
- Limitiert: 2.000-10.000 Stück
- Reguläre Auflage: > 10.000 Stück (dann meist nicht angegeben)

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – freie Eingabe als Ganzzahl.
