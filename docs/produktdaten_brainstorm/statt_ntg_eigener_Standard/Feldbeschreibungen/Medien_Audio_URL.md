# Audio_URL

## Feldgruppe
Medien

## Beschreibung
Die URL zu einer Audiodatei – typischerweise Sound-Samples bei Sound-Lokomotiven.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Audio_URL |
| Datentyp | URL |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 500 |
| Format | https://... |

## Hinweis für Datenlieferanten
Audiodateien sind besonders relevant bei Sound-Lokomotiven:

**Empfohlener Inhalt:**
- Motorgeräusch / Dampfschlag
- Pfeife / Horn
- Ansagen
- Fahrgeräusche

**Technische Anforderungen:**
- Format: MP3 oder WAV
- Qualität: Mindestens 192 kbps
- Länge: 10-60 Sekunden

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – URL vom Hersteller.

## Beispiele
```
https://www.maerklin.de/media/sound/39212_sound_sample.mp3
https://cdn.hersteller.de/audio/product_12345.wav
```
