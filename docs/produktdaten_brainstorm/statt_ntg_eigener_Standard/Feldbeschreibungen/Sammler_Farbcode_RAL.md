# Farbcode_RAL

## Feldgruppe
Sammlermodelle

## Beschreibung
Der RAL-Farbcode der Modellfarbe für eindeutige Farbzuordnung.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Farbcode_RAL |
| Datentyp | Text |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 10 |
| Format | RAL XXXX oder XXXX |

## Hinweis für Datenlieferanten
Der RAL-Code ermöglicht eine eindeutige Farbidentifikation, besonders bei:
- Nutzfahrzeugen (Feuerwehr, THW, Polizei)
- Militärfahrzeugen
- Baumaschinen
- Schienenfahrzeugen

**Hinweis:** Viele Automobilfarben haben keinen RAL-Entsprechung, da Hersteller eigene Farbcodes verwenden.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Standardisierte RAL-Farben. Häufig verwendete Codes:

| RAL | Farbname | Anwendung |
|-----|----------|-----------|
| 3000 | Feuerrot | Feuerwehr |
| 3020 | Verkehrsrot | LKW, Busse |
| 5002 | Ultramarinblau | THW |
| 5005 | Signalblau | Polizei (alt) |
| 6003 | Olivgrün | Bundeswehr |
| 6014 | Gelboliv | NVA |
| 7012 | Basaltgrau | Bundeswehr |
| 9010 | Reinweiß | Lieferwagen |
