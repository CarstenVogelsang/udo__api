# Video_URL

## Feldgruppe
Medien

## Beschreibung
Die URL zu einem Produktvideo.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Video_URL |
| Datentyp | URL |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 500 |
| Format | https://... |

## Hinweis für Datenlieferanten
Produktvideos zeigen das Produkt in Aktion:

**Empfohlener Inhalt:**
- Fahrt auf Anlage (Loks, Wagen)
- Funktionsvorführung (Sound, Licht, Rauch)
- Aufbauanleitung (Bausätze)
- Detailansichten

**Technische Anforderungen:**
- Mindestauflösung: 1080p (Full HD)
- Format: MP4 oder YouTube/Vimeo-Link
- Länge: 30 Sekunden bis 3 Minuten

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – URL vom Hersteller.

## Beispiele
```
https://www.youtube.com/watch?v=XXXXXXXXX
https://vimeo.com/123456789
https://cdn.hersteller.de/videos/product_12345.mp4
```

## Hinweis
YouTube-Links werden bevorzugt, da sie auf allen Plattformen eingebettet werden können.
