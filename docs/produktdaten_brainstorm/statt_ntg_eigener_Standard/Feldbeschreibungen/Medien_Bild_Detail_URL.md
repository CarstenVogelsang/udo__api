# Bild_Detail_URL

## Feldgruppe
Medien

## Beschreibung
Die URL zu einem Detailbild – zeigt besondere Merkmale oder Details des Produkts.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Bild_Detail_URL |
| Datentyp | URL |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 500 |
| Format | https://... |

## Hinweis für Datenlieferanten
Detailbilder zeigen besondere Aspekte des Produkts:

**Typische Detailbilder:**
- Nahaufnahme von Beschriftungen
- Führerstand-Innenraum (Loks)
- Geöffnete Türen/Motorhaube (Sammlermodelle)
- Feine Gravuren oder Bedruckungen
- Besondere Ausstattungsmerkmale

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – URL vom Hersteller.

## Beispiele
```
https://www.maerklin.de/media/product/39212_detail_1.jpg
https://cdn.hersteller.de/images/products/12345_closeup.jpg
```

## Mehrere Detailbilder
Wenn mehrere Detailbilder verfügbar sind, können diese durch Semikolon getrennt werden:
```
https://url1.jpg;https://url2.jpg;https://url3.jpg
```
