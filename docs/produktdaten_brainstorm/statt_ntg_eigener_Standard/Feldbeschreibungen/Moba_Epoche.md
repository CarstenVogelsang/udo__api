# Epoche

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Die historische Eisenbahnepoche, der das Modell zugeordnet ist – bestimmt die zeitliche Einordnung des Vorbilds.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Epoche |
| Datentyp | Text (Code) |
| Pflichtfeld | Empfohlen |
| Zeichenlänge | 3-5 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die Epocheneinteilung ist für Modellbahner wichtig, um zeitlich passende Fahrzeuge zusammenzustellen.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Mehrfach-Epochen
Manche Modelle passen in mehrere Epochen:
- "III/IV" = Übergangszeit
- "V-VI" = Von Epoche V bis VI einsetzbar
