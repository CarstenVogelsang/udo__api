# Edition

## Feldgruppe
Sammlermodelle

## Beschreibung
Die Edition oder Sonderserie, zu der das Modell gehört.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Edition |
| Datentyp | Text |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 100 |

## Hinweis für Datenlieferanten
Die Edition bezeichnet besondere Serien oder Sonderauflagen:

**Typische Editionen:**
- Jubiläumsausgaben ("50 Jahre Porsche 911")
- Sonderserien ("Signature Series", "First Edition")
- Händler-Exklusiv ("Dealer Edition")
- Club-Modelle ("Club Model 2024")
- Messe-Sondermodelle ("Spielwarenmesse 2024")

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
