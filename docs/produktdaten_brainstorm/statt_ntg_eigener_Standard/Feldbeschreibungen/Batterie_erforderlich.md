# Batterie_erforderlich

## Feldgruppe
Batterie

## Beschreibung
Angabe, ob das Produkt Batterien für den Betrieb benötigt.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Batterie_erforderlich |
| Datentyp | Text (Code) |
| Pflichtfeld | Ja |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Diese Angabe ist für Kunden wichtig, um zu wissen, ob beim Kauf zusätzlich Batterien benötigt werden.

**Batterie erforderlich (J) bei:**
- Batteriebetriebene Fahrzeuge
- Fernbedienungen
- Sound-Module
- Beleuchtete Sammlermodelle

**Batterie nicht erforderlich (N) bei:**
- Schienengebundene Modelle (Stromversorgung über Gleis)
- Reine Bausätze ohne Elektronik
- Mechanische Modelle

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Batterien für Betrieb erforderlich |
| N | Keine Batterien erforderlich |

## Beispiele
| Produkt | Batterie_erforderlich |
|---------|----------------------|
| Batteriebetriebene Kinderbahn | J |
| Fernbedienung für Digital | J |
| Sammlermodell mit LED | J |
| Standard H0-Lokomotive | N |
| Gebäudebausatz ohne Licht | N |
