# Beleuchtung_Sammler

## Feldgruppe
Sammlermodelle

## Beschreibung
Angabe, ob das Sammlermodell über eine funktionsfähige Beleuchtung verfügt.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Beleuchtung_Sammler |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 3 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Beleuchtung bei Sammlermodellen ist ein besonderes Feature, das den Vitrinenwert erhöht.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Beleuchtung | Beschreibung |
|------|-------------|--------------|
| LED | LED | LED-Beleuchtung (Scheinwerfer, Rücklichter) |
| BAT | Batterie | Batteriebetriebene Beleuchtung |
| USB | USB | USB-betriebene Beleuchtung |
| KEI | Keine | Keine Beleuchtung |

## Beispiele
| Hersteller | Beleuchtung_Sammler |
|------------|---------------------|
| AUTOart (Signature) | LED |
| Minichamps | KEI |
| CMC | KEI |
| Diecast-Masters | LED (LKW-Modelle) |
