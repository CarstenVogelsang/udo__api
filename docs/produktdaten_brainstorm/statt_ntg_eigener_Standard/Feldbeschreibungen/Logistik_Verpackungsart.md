# Verpackungsart

## Feldgruppe
Logistik & Verpackung

## Beschreibung
Die Art der Verkaufsverpackung, in der das Produkt an den Endkunden geliefert wird.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Verpackungsart |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 3 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die Verpackungsart ist relevant für:
- Lagerung (z.B. hängende vs. liegende Lagerung)
- Produktpräsentation im Handel
- Umwelt- und Verpackungskennzeichnung (LUCID)
- Retourenhandling

Bitte verwenden Sie ausschließlich die Codes aus der zugehörigen Werteliste.

Bei Änderungen der Verpackungsart informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Siehe **Werteliste_Logistik_Verpackungsart.xlsx**

| Code | Verpackungsart | Beschreibung |
|------|----------------|--------------|
| KAR | Karton | Standard-Faltkarton |
| BLI | Blister | Sichtverpackung mit Kunststoffhaube |
| VIT | Vitrinenbox | Transparente Kunststoffbox |
| SET | Setbox | Großkarton für Sets/Packungen |
| FOL | Folienbeutel | Polybeutel, oft für Kleinteile |
| SKK | Steckkarte | Karton mit Aufhängeloch |
| SCH | Schachtel | Stülpdeckelschachtel |
| OVP | Originalverpackung | Herstellerspezifische Verpackung |
