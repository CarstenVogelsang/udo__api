# Bild_Anwendung_URL

## Feldgruppe
Medien

## Beschreibung
Die URL zu einem Anwendungs- oder Ambientebild – zeigt das Produkt im Einsatz oder in einer Szene.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Bild_Anwendung_URL |
| Datentyp | URL |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 500 |
| Format | https://... |

## Hinweis für Datenlieferanten
Anwendungsbilder zeigen das Produkt in realistischer Umgebung:

**Typische Anwendungsbilder:**
- Lokomotive auf Modellbahnanlage
- Gebäude in gestalteter Landschaft
- Fahrzeug auf Diorama
- Zug in Aktion (Bewegungsunschärfe)

**Vorteile:**
- Inspiriert den Kunden
- Zeigt maßstäbliche Proportionen
- Demonstriert Einsatzmöglichkeiten

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – URL vom Hersteller.
