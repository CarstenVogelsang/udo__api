# Spurweite

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Die Nenngröße/Spurweite des Modells – bestimmt die Kompatibilität mit Gleissystemen und anderen Fahrzeugen.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Spurweite |
| Datentyp | Text (Code) |
| Pflichtfeld | Ja (bei Rollmaterial) |
| Zeichenlänge | 3 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die Spurweite ist das wichtigste Kompatibilitätskriterium bei Modellbahnen. Falsche Angaben führen zu Reklamationen.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
