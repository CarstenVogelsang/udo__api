# Federung

## Feldgruppe
Sammlermodelle

## Beschreibung
Angabe, ob das Modell über eine funktionsfähige Federung verfügt.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Federung |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Eine funktionsfähige Federung ist ein Luxusmerkmal bei hochwertigen Sammlermodellen (CMC, AUTOart etc.).

**Typische Ausführungen:**
- Federnde Achsen
- Bewegliche Aufhängung
- Nachbildung des Originalfahrwerks

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Funktionsfähige Federung vorhanden |
| N | Keine Federung |
