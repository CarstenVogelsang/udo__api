# Bahngesellschaft

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Die Bahngesellschaft, deren Lackierung und Beschriftung das Modell trägt.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Bahngesellschaft |
| Datentyp | Text (Code) |
| Pflichtfeld | Empfohlen |
| Zeichenlänge | 3-5 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die Bahngesellschaft ist wichtig für die Zuordnung zu nationalen Vorlieben und historischen Perioden.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
