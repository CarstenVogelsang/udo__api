# Motorhaube_oeffenbar

## Feldgruppe
Sammlermodelle

## Beschreibung
Angabe, ob die Motorhaube des Modells zu öffnen ist.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Motorhaube_oeffenbar |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Eine öffenbare Motorhaube zeigt den detaillierten Motorraum – ein wichtiges Qualitätsmerkmal bei hochwertigen Sammlermodellen.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Motorhaube ist zu öffnen |
| N | Motorhaube ist nicht zu öffnen |

## Hinweis
Bei Fahrzeugen mit Heckmotor (z.B. Porsche 911, VW Käfer) bezieht sich "Motorhaube" auf die hintere Abdeckung. Die vordere Haube wird über Heckklappe_oeffenbar (als Kofferraumdeckel) erfasst.
