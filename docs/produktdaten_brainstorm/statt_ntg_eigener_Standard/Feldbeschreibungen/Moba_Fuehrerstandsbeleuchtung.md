# Fuehrerstandsbeleuchtung

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Angabe, ob das Modell eine beleuchtete Führerstandskabine hat.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Fuehrerstandsbeleuchtung |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die Führerstandsbeleuchtung ist ein Premium-Feature, das bei hochwertigen Modellen den Führerstand von innen beleuchtet.

**Typische Umsetzung:**
- LED im Führerstand
- Schaltbar per Digitalfunktion
- Oft mit Lokführer-Figur kombiniert

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Führerstandsbeleuchtung vorhanden |
| N | Keine Führerstandsbeleuchtung |

## Beispiele
| Produkt | Führer­stands­beleuchtung | Hinweis |
|---------|-------------------------|---------|
| Märklin 39212 (Premium) | J | Digital schaltbar |
| Märklin 36183 (Standard) | N | Nicht vorhanden |
| ESU BR 218 (Premium) | J | Mit Lokführer-Figur |

## Hinweis
Bei Digital-Modellen wird die Führerstandsbeleuchtung oft über eine separate Funktionstaste (z.B. F7 oder F8) gesteuert.
