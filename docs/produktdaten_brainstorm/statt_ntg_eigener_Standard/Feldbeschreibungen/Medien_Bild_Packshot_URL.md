# Bild_Packshot_URL

## Feldgruppe
Medien

## Beschreibung
Die URL zum Packshot – das Bild der Verkaufsverpackung.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Bild_Packshot_URL |
| Datentyp | URL |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 500 |
| Format | https://... |

## Hinweis für Datenlieferanten
Der Packshot zeigt die Original-Verkaufsverpackung:

**Bildanforderungen:**
- Mindestauflösung: 1500 x 1500 Pixel
- Format: JPG oder PNG
- Hintergrund: Weiß oder freigestellt
- Ansicht: Frontseite der Verpackung

**Relevanz:**
- Kunden erkennen die Verpackung im Handel
- Geschenkeignung bewerten
- Originalität verifizieren

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – URL vom Hersteller.

https://cdn.hersteller.de/images/products/12345_box.png
