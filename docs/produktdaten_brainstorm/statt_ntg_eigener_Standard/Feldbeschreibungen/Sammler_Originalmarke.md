# Originalmarke

## Feldgruppe
Sammlermodelle

## Beschreibung
Die Marke des Originalfahrzeugs, das als Vorbild für das Modell diente.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Originalmarke |
| Datentyp | Text |
| Pflichtfeld | Empfohlen |
| Zeichenlänge | max. 50 |

## Hinweis für Datenlieferanten
Die Originalmarke bezeichnet den Hersteller des Vorbildfahrzeugs, nicht den Modellhersteller.

**Wichtig:** Markenrechtlich korrekte Schreibweise verwenden.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
