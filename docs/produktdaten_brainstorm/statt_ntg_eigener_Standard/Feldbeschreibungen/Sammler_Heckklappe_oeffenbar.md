# Heckklappe_oeffenbar

## Feldgruppe
Sammlermodelle

## Beschreibung
Angabe, ob die Heckklappe oder der Kofferraumdeckel des Modells zu öffnen ist.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Heckklappe_oeffenbar |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die öffenbare Heckklappe ermöglicht den Blick in den Kofferraum oder Laderaum.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Heckklappe/Kofferraum ist zu öffnen |
| N | Heckklappe ist nicht zu öffnen |
