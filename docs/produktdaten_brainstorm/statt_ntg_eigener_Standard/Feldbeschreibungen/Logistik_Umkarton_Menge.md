# Umkarton_Menge

## Feldgruppe
Logistik & Verpackung

## Beschreibung
Die Anzahl der Verkaufseinheiten (VKE), die in einem Umkarton/Masterpack enthalten sind.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Umkarton_Menge |
| Datentyp | Ganzzahl |
| Pflichtfeld | Nein |
| Minimalwert | 1 |
| Maximalwert | 9999 |

## Hinweis für Datenlieferanten
Die Umkarton-Menge gibt an, wie viele Einzelverpackungen eines Artikels in einem Versandkarton zusammengefasst werden.

**Bedeutung für den Handel:**
- Bestellmengen orientieren sich oft an Umkarton-Mengen
- Rabattstaffeln können auf Umkarton-Mengen basieren
- Lagerplanung berücksichtigt Gebindegrößen

Typische Umkarton-Mengen im Modellbahnhandel:
- Hochpreisige Lokomotiven: 1-3 Stück
- Wagen: 6-12 Stück
- Zubehör: 6-24 Stück
- Kleinteile: 12-48 Stück

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
