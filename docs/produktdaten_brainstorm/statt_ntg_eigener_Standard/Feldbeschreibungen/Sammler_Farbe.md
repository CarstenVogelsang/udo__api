# Farbe

## Feldgruppe
Sammlermodelle

## Beschreibung
Die Farbbezeichnung des Sammlermodells.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Farbe |
| Datentyp | Text |
| Pflichtfeld | Empfohlen |
| Zeichenlänge | max. 50 |

## Hinweis für Datenlieferanten
Die Farbbezeichnung sollte möglichst die Original-Farbbezeichnung des Vorbilds verwenden.

**Empfohlene Angaben:**
- Originalfarbname (z.B. "Papyrus", "Graphitgrau metallic")
- Alternativ: Beschreibende Farbe (z.B. "Silber metallic")
- Bei Mehrfarbigkeit: Hauptfarbe + Akzentfarbe

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – freier Text, orientiert an Originalfarbbezeichnungen.

## Beispiele
| Modell | Farbe |
|--------|-------|
| Mercedes 300 SL | Silber metallic |
| Porsche 911 | Guards Red |
| VW Käfer | Pastelweiß (L90D) |
| Ferrari F40 | Rosso Corsa |
| BMW M3 | Dakar Gelb |
| MAN TGX | Verkehrsrot RAL 3020 |

## Zusammenhang
Bei Bedarf kann die Farbe zusätzlich über das Feld Farbcode_RAL präzisiert werden.
