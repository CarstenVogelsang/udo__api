# Batterien_austauschbar

## Feldgruppe
Batterie

## Beschreibung
Angabe, ob die Batterien vom Endkunden selbst ausgetauscht werden können.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Batterien_austauschbar |
| Datentyp | Text (Code) |
| Pflichtfeld | Bedingt (wenn Batterie_erforderlich = J) |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Diese Information ist wichtig für:
- Kundenerwartung (Lebensdauer des Produkts)
- Batteriegesetz (Hinweispflichten)
- Nachhaltigkeitsbewertung

**Batterien austauschbar (J):**
- Batteriefach mit Deckel
- Ohne Werkzeug zugänglich
- Standardbatterien

**Batterien nicht austauschbar (N):**
- Fest verbaute Akkus
- Verschweißte Gehäuse
- Spezialakkus (Fachhandel-Service)

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Batterien vom Kunden austauschbar |
| N | Batterien nicht austauschbar (fest verbaut) |

## Beispiele
| Produkt | Batterien_austauschbar |
|---------|------------------------|
| Fernbedienung | J |
| Kinderbahn | J |
| Modell mit Akku | N |
| Beleuchtetes Gebäude | J |

## Rechtlicher Hinweis
Bei nicht austauschbaren Batterien sind besondere Entsorgungshinweise erforderlich.
