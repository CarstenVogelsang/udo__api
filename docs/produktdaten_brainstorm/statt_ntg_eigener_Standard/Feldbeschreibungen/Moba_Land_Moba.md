# Land_Moba

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Das Land, in dem das Vorbild des Modells eingesetzt wurde/wird – ergänzt die Bahngesellschaft für die Shop-Filterung.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Land_Moba |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 2 |
| Format | ISO 3166-1 Alpha-2 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Land_Moba bezeichnet das Einsatzland des Vorbilds, nicht das Herstellungsland des Modells.

Mehrländer-Einsatz (z.B. ÖBB-Taurus auch in Deutschland) wird mit dem Haupteinsatzland angegeben.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
