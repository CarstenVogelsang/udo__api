# Batterie_im_Lieferumfang

## Feldgruppe
Batterie

## Beschreibung
Angabe, ob die benötigten Batterien im Lieferumfang enthalten sind.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Batterie_im_Lieferumfang |
| Datentyp | Text (Code) |
| Pflichtfeld | Bedingt (wenn Batterie_erforderlich = J) |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Diese Information ist wichtig für:
- Kundenerwartung (direkt einsatzbereit oder nicht?)
- Batteriegesetz-Registrierung (BattG)
- Verpackungskennzeichnung

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Batterien sind im Lieferumfang enthalten |
| N | Batterien müssen separat erworben werden |

## Beispiele
| Produkt | Batterie_erforderlich | Batterie_im_Lieferumfang |
|---------|----------------------|--------------------------|
| Kinderbahn-Startset | J | J |
| Fernbedienung | J | N |
| Sound-Modul | J | N |

## Rechtlicher Hinweis
Wenn Batterien im Lieferumfang enthalten sind, muss der Hersteller im Batterieregister (LUCID) registriert sein.
