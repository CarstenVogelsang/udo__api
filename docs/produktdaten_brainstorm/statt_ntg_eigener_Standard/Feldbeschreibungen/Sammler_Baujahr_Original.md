# Baujahr_Original

## Feldgruppe
Sammlermodelle

## Beschreibung
Das Baujahr oder der Bauzeitraum des Originalfahrzeugs.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Baujahr_Original |
| Datentyp | Text |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 20 |
| Format | JJJJ oder JJJJ-JJJJ |

## Hinweis für Datenlieferanten
Das Baujahr hilft Sammlern bei der zeitlichen Einordnung und beim Aufbau epochengerechter Sammlungen.

**Mögliche Formate:**
- Einzeljahr: "1963"
- Zeitraum: "1963-1971"
- Modelljahr: "1965 (MY)"
- Ab/Bis: "ab 1998"

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
