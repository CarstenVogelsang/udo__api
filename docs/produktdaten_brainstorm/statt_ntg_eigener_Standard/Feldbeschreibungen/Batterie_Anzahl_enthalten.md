# Batterie_Anzahl_enthalten

## Feldgruppe
Batterie

## Beschreibung
Die Anzahl der im Lieferumfang enthaltenen Batterien.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Batterie_Anzahl_enthalten |
| Datentyp | Ganzzahl |
| Pflichtfeld | Bedingt (wenn Batterie_im_Lieferumfang = J) |
| Minimalwert | 0 |
| Maximalwert | 99 |

## Hinweis für Datenlieferanten
Diese Angabe ist wichtig für:
- Batteriegesetz-Meldung (BattG)
- Kundeninformation

Bei Batterie_im_Lieferumfang = N wird 0 oder leer eingetragen.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – freie Eingabe als Ganzzahl.

## Beispiele
| Produkt | Batterie_im_Lieferumfang | Anzahl_enthalten |
|---------|--------------------------|------------------|
| Kinderbahn-Startset | J | 4 |
| Fernbedienung | N | 0 |
| Demo-Modell | J | 2 |
