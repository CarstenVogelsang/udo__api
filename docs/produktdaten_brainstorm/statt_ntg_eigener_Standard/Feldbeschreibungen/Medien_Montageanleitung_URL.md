# Montageanleitung_URL

## Feldgruppe
Medien

## Beschreibung
Die URL zur Montageanleitung oder Bauanleitung als PDF.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Montageanleitung_URL |
| Datentyp | URL |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 500 |
| Format | https://... |

## Hinweis für Datenlieferanten
Die Montageanleitung ist besonders wichtig bei:
- Bausätzen (Gebäude, Fahrzeuge)
- Nachrüstsätzen (Decoder, Beleuchtung)
- Zubehör mit Einbauhinweisen

**Anforderungen:**
- Format: PDF
- Sprache: Mindestens Deutsch
- Auflösung: Gut lesbar, druckfähig

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – URL vom Hersteller.

## Beispiele
```
https://www.noch.de/downloads/8904_anleitung.pdf
https://cdn.hersteller.de/docs/product_12345_manual_de.pdf
```
