# Rauchsatz_Artikelnr

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Die Artikelnummer des passenden Rauchsatz-Nachrüstsatzes für Modelle mit Rauchsatz-Vorbereitung.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Rauchsatz_Artikelnr |
| Datentyp | Text |
| Pflichtfeld | Nein |
| Zeichenlänge | max. 20 |

## Hinweis für Datenlieferanten
Dieses Feld gibt dem Kunden einen direkten Hinweis auf das passende Zubehör:

**Voraussetzung:**
- Rauchsatz_Vorbereitung = J
- Passender Nachrüstsatz verfügbar

**Inhalt:**
- Artikelnummer des Nachrüstsatzes
- Vom gleichen Hersteller

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Keine – Artikelnummer des Herstellers.

## Beispiele
| Modell | Rauchsatz_Artikelnr | Hersteller |
|--------|---------------------|------------|
| Märklin BR 01 | 72270 | Märklin |
| Märklin BR 44 | 72270 | Märklin |
| Märklin BR 50 | 72270 | Märklin |
| ESU Dampflok | 51900 | ESU |

## Hinweis
Die Artikelnummer ermöglicht dem Shop-System eine automatische Zubehör-Empfehlung auf der Produktseite.
