# Laenge_ueber_Puffer_mm

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Die Gesamtlänge des Modells in Millimetern, gemessen über die Puffer (LüP).

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Laenge_ueber_Puffer_mm |
| Datentyp | Ganzzahl |
| Pflichtfeld | Empfohlen |
| Einheit | Millimeter (mm) |
| Minimalwert | 10 |
| Maximalwert | 9999 |

## Hinweis für Datenlieferanten
Die Länge über Puffer (LüP) ist die wichtigste Maßangabe für Schienenfahrzeuge:

**Messmethode:**
- Von Pufferbohle zu Pufferbohle
- Bei eingerasteten/montierten Kupplungen
- Horizontale Messung

**Relevanz:**
- Gleisplanung (Bahnsteiglängen, Abstellgleise)
- Vitrinenplanung für Sammler
- Maßstabsberechnung

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Maßstabsumrechnung
```
LüP Modell = LüP Vorbild / Maßstab

Beispiel BR 103:
Vorbild: 19.500 mm
Maßstab: 1:87
Modell: 19.500 / 87 ≈ 224 mm
```
