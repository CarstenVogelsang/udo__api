# Innenbeleuchtung

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Angabe, ob das Modell eine Innenbeleuchtung hat (typisch bei Personenwagen und Triebwagen).

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Innenbeleuchtung |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 3 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die Innenbeleuchtung wertet Personenwagen und Triebzüge optisch auf und ist ein beliebtes Feature.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
Siehe **Werteliste_Moba_Innenbeleuchtung.xlsx**

| Code | Innenbeleuchtung | Beschreibung |
|------|------------------|--------------|
| LED | LED eingebaut | LED-Innenbeleuchtung ab Werk |
| GLU | Glühlampe | Glühlampen-Innenbeleuchtung |
| KEI | Keine | Keine Innenbeleuchtung |
| VOR | Vorbereitet | Vorbereitet für Nachrüstung |
| NAC | Nachrüstbar | Nachrüstsatz verfügbar |

## Beispiele
| Produkt | Innenbeleuchtung | Hinweis |
|---------|------------------|---------|
| Märklin IC-Wagen | LED | LED-Beleuchtung ab Werk |
| Märklin Eilzugwagen | VOR | Schnittstelle vorhanden |
| Fleischmann Wagen | NAC | Nachrüstsatz erhältlich |
| Güterwagen | KEI | Nicht relevant |

## Stromversorgung
Die Innenbeleuchtung wird gespeist durch:
- Radschleifer (AC-System)
- Stromführende Kupplung
- Batterie/Kondensator
- Induktiv (selten)

## Hinweis zur Nachrüstung
Bei "VOR" (Vorbereitet) ist oft eine Artikelnummer für den Nachrüstsatz in der Zubehör-Dokumentation angegeben.
