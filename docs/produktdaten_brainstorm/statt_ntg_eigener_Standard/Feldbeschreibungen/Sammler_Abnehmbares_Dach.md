# Abnehmbares_Dach

## Feldgruppe
Sammlermodelle

## Beschreibung
Angabe, ob das Modell ein abnehmbares Dach oder Hardtop hat.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Abnehmbares_Dach |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Relevant bei:
- Cabrios mit Hardtop
- Targa-Modellen
- Fahrzeugen mit abnehmbarem Verdeck

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Dach ist abnehmbar |
| N | Dach ist fest / kein Dach vorhanden |
