# Qualitaetsstufe

## Feldgruppe
Sammlermodelle

## Beschreibung
Die Qualitätsstufe des Sammlermodells – von Spielzeugqualität bis hin zu Museumsqualität.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Qualitaetsstufe |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 3 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Die Qualitätsstufe gibt dem Kunden eine schnelle Orientierung über die zu erwartende Detaillierung und Verarbeitung.

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
