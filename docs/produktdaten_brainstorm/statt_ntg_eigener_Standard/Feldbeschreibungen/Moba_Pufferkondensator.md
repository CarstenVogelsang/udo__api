# Pufferkondensator

## Feldgruppe
Moba Rollmaterial

## Beschreibung
Angabe, ob das Modell einen eingebauten Pufferkondensator (Energiespeicher) hat, der Stromunterbrechungen überbrückt.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Pufferkondensator |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Der Pufferkondensator ist ein wichtiges Qualitätsmerkmal für digitale Modellbahnen:

**Vorteile eines Pufferkondensators:**
- Überbrückt kurze Stromunterbrechungen (z.B. an Weichen)
- Verhindert Soundaussetzer bei Sound-Modellen
- Verbessert das Langsamfahrverhalten
- Erhöht die Betriebssicherheit

**Typische Ausstattung:**
- Hochwertige Loks mit Sound: meist J
- Einsteigermodelle: meist N
- Wagen mit Beleuchtung: selten

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
