# Material

## Feldgruppe
Sammlermodelle

## Beschreibung
Das Hauptmaterial des Sammlermodells – wichtig für Qualitätseinstufung und Werterhalt.

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Material |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 3 |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Das Material ist ein wichtiges Qualitätsmerkmal für Sammler:

**Materialien nach Wertigkeit:**
1. Zinkdruckguss (Diecast) – hochwertig, schwer
2. Resin – hochdetailliert, limitiert
3. Metall-Kunststoff-Mix – gutes Preis-Leistungs-Verhältnis
4. Kunststoff – Einsteigerklasse

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.
