# Lenkung_funktionsfaehig

## Feldgruppe
Sammlermodelle

## Beschreibung
Angabe, ob die Lenkung des Modells funktionsfähig ist (Räder lassen sich lenken).

## Technische Spezifikation
| Eigenschaft | Wert |
|-------------|------|
| Feldname | Lenkung_funktionsfaehig |
| Datentyp | Text (Code) |
| Pflichtfeld | Nein |
| Zeichenlänge | 1 |
| Erlaubte Werte | J, N |
| Groß-/Kleinschreibung | Egal (wird normalisiert) |

## Hinweis für Datenlieferanten
Eine funktionsfähige Lenkung ist ein Premium-Feature bei hochwertigen Sammlermodellen.

**Typische Ausführungen:**
- Lenkrad mit Lenkstange verbunden
- Vorderräder einschlagbar
- Oft in Kombination mit beweglichen Türen

Bei Änderungen informieren Sie MHI proaktiv durch Einsendung einer aktualisierten Gesamt- oder Änderungsdatei.

## Werteliste
| Code | Bedeutung |
|------|-----------|
| J | Lenkung ist funktionsfähig |
| N | Keine funktionsfähige Lenkung |
